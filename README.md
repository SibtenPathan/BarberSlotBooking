# Barber Slot Booking System

A full-stack mobile application for booking barber appointments built with React Native (Expo) and Node.js.

## Project Structure

```
BarberSlotBooking/
├── frontend/          # React Native Expo application
├── backend/           # Node.js Express API server
└── README.md         # This file
```

## Quick Start

### Backend Setup

```bash
cd backend
npm install
# Create .env file with your configuration
npm start
```

### Frontend Setup

```bash
cd frontend
npm install
npx expo start
```

## Features

- User authentication and authorization
- Barber shop browsing and details
- Appointment booking system
- Payment history tracking
- In-app messaging
- Push notifications
- User profile management

## Technology Stack

### Frontend
- React Native with Expo
- TypeScript
- NativeWind (Tailwind CSS)
- React Navigation
- Expo Router

### Backend
- Node.js
- Express.js
- MongoDB with Mongoose
- JWT Authentication
- Socket.io for real-time features
- Cloudinary for image uploads
